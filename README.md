# MovieHub

A simple website to watch and stream a collection of movies, with a special focus on Doraemon!

**Live Site:** [https://doraemonhub.netlify.app/](https://doraemonhub.netlify.app/)

## Getting Started

To run this project locally, you'll need to serve it with a local web server due to browser security policies for JavaScript modules.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/nazib10/moviehub.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd moviehub
    ```
3.  **Start a local web server.** If you have Python 3 installed, you can run:
    ```bash
    python -m http.server
    ```
    If you have Python 2, run:
    ```bash
    python -m SimpleHTTPServer
    ```
4.  **Open your web browser** and go to `http://localhost:8000`.
